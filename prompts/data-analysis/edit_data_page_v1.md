## Purpose
Add an Edit Data page with full CRUD functionality for the dataset.

## Prompt
Add Edit Data page, connected with an Edit Data button at the bottom of the visual, to support CRUD operations on the entire full dataset.

The Edit Data page should include:
- The ability to modify the list of valid sectors
- The ability to download the full dataset as a CSV file
- The ability to upload new data from a CSV file

Do not change any other application functionality.
