# Prompt: edit_data_page
version: v2
status: experimental
model: gemini

## Purpose
Add an EDIT Data page with full CRUD functionality for the dataset.

## Prompt
Add an Edit Data page, connected to an Edit Data button at the bottom of all application pages, to support CRUD operations on the entire full dataset.

The Edit Data page should also include:
- The ability to add new records
- The ability to delete records
- The ability to download the full dataset as a CSV file
- The ability to upload new data from a CSV file

Do not change any other application functionality.

## Notes
- Ensure existing application functionality remains unchanged
- Maintain compatibility with the current data structure
